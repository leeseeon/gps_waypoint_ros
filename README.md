DS LAB 프로젝트는 **다중 UGV(무인 지상 차량)**와 GCS(Ground Control Station) 간의 실시간 양방향 통신을 구현하기 위한 연구 프로젝트입니다.
ROS 기반 자율주행 시스템과 Node.js/React 기반 웹 GCS를 연동하여, GPS·IMU·LiDAR 데이터를 실시간 스트리밍하고, TCP/UDP 프로토콜을 이용한 저지연 제어 구조를 설계하였습니다.
또한 다중 로봇 환경에서의 동기화 문제, 네트워크 지연 보정, 상태 관리 아키텍처 설계를 통해 확장 가능한 UGV 통합 관제 시스템을 구현하는 것을 목표로 합니다.
